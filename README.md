# Heart-Disease-Prediction
using of machine learning to predict whether a person has heart disease based on medical data
## Description
Heart Disease Prediction is a machine learning project designed to predict whether a person is likely to have heart disease based on medical attributes such as age, cholesterol level, blood pressure, and other health indicators. The system analyzes patient data and provides an early prediction, helping in preventive healthcare and decision-making.
## Technologies used
-python
-pandas
-scikit-learn
-matplotlip
## Features
-Data preprocessing (handling missing values, normalization)
-Exploratory Data Analysis (EDA)
-Feature selection & importance analysis
-Multiple ML model training
-Model evaluation using accuracy, precision, recall
-Prediction system for new patient data
-Visualization of trends and correlations
## Goal
Goal
-The main goal of this project is:
-To build an accurate and reliable machine learning model
-To detect heart disease at an early stage
-To assist doctors and healthcare systems in decision-making
-To reduce risk by providing quick predictions
## Dataset
-The dataset contains medical records of patients
-Common features include:
-Age
-Sex
-Chest pain type
-Resting blood pressure
-Cholesterol level
-Fasting blood sugar
-ECG results
-Maximum heart rate
-Exercise-induced angina
## Target variable:
-0 → No Heart Disease
-1 → Heart Disease
## Models Used
-Logistic Regression
-Decision Tree Classifier
-Random Forest Classifier
-Support Vector Machine (SVM)
## Output
-The model predicts whether a person has heart disease or not
- I applied some machine learning algorithms are logistic regession , naive bayes , svm, k nearest neighbors , neural network , random forest , XG boost , decision tree.i varied parameters across each model to improve their scores,finally Decision Tree achieved the the heighest score of 80.56%.
